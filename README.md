# AlgoFlow

AlgoFlow is a smooth, minimal DSA visualizer with animations, pseudocode, and real code side-by-side.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm start
```

## Project layout

- `src/app/` – routes and layouts (App Router)
- `src/components/` – reusable UI components
- `src/lib/algorithms.ts` – algorithm metadata + steps (single source of truth)
